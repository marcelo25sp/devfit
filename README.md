# 🏋️ DevFit

Sistema de gestão para academias desenvolvido com base na disciplina de Fundamentos de Engenharia de Software.

Sistema desenvolvido com Java + Spring Boot.

## 🚀 Objetivo
Projeto prático para aplicar conceitos de Engenharia de Software, incluindo requisitos, modelagem e desenvolvimento backend.

- Dia 2: Documento de modelagem do sistema criado
